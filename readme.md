# is-osx [![Build Status](https://travis-ci.org/skippednote/is-osx.svg?branch=master)](https://travis-ci.org/skippednote/is-osx)

> Check if the operating system is OSX


## Install

```
$ npm install --save is-osx
```


## Usage

```js
var isOsx = require('is-osx');

isOsx()
//=> true
```
